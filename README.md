# 📝 ToDo App

A powerful and evolving To-Do management application built with Kotlin, designed to help users plan, prioritize, and understand their productivity.

This project has grown from a simple task manager into a **productivity-focused system** with structured task categorization and real-time tracking.

## 🔹 Version 1.0 — Foundation Release

### ✨ Core Features
- Task creation, editing, and deletion  
- Simple and minimal UI  
- Basic task organization  

### 🧠 Eisenhower Matrix (Urgency × Importance)

Tasks were categorized into:

| Quadrant | Category    | Description                |
|----------|------------|----------------------------|
| Q1       | Do Now     | Urgent & Important         |
| Q2       | Plan       | Not Urgent but Important   |
| Q3       | Delegate   | Urgent but Not Important   |
| Q4       | Eliminate  | Not Urgent & Not Important |

This helped users prioritize tasks effectively.
